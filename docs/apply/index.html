Run the MemSQL Docker Image

The MemSQL Docker image includes a single-node MemSQL cluster with one master agregator and one leaf, along with a benchmarking program to test MemSQL performance. For more information on MemSQL’s architecture, refer to MemSQL’s Concepts Overview.

Before you run the Docker image, first you need to verify your system requirements. From Docker, type the following:

$ docker run --rm memsql/quickstart check-system

If the MemSQL system test fails, refer to Requirements for more information and additional installation options.
