Install Docker

Before you begin, make sure to save all of your work and close all of your apps. You’ll need to sign out of Windows when the Docker installation completes. If Hyper-V is not enabled, you’ll also need to restart your PC.

To install Docker Community Edition for Windows:

  1. Navigate to the Docker Community Edition for Windows download page.
  2. Click “Please Login To Download.”
  3. On the “Welcome to Docker” page, login with your Docker ID or create a new account.
  4. If you created a new account, check your email and click the link provided to verify your email address. This should return you to the Docker Community Edition for Windows download page. Now that your account is verified, you’ll need to login to enable the download.
  5. Once you’re logged in, click “Get Docker” to download the installation file from the Docker Community Edition for Windows download page.
  6. After the installation file downloads, run “Docker for Windows Installer.exe” from your browser.
  7. On the Configuration window, do not check “Use Windows containers instead of Linux containers.” You’ll need a Linux environment to run MemSQL. Refer to Get started with Docker for Windows - Switch between Windows and Linux containers for more information. Click “Ok” to continue.
  8. Docker will start “Unpacking files.” On the “Installation succeeded” window, click “Close and log out.” This will sign you out of Windows.
  9. If “Hyper-V and Containers features are not enabled,” click “Ok.” Windows will enable these features, and your PC will automatically restart. This step can take a few minutes to complete.
  10. When you sign back in, Docker should be up and running. On the Welcome window near your system tray, enter your Docker ID and Password, then click Log In. If Docker doesn’t start automatically, refer to Start Docker for Windows for more information.
