Docker Overview

Docker is a software platform that enables you to build, run, and install application packages called containers. Containers include everything you need to run an app inside the Docker platform, and do not rely on any external runtimes or files located on your operating system. Container images, like the MemSQL image, are layered templates that become containers when you run them on the Docker Engine. Refer to Docker’s What is a Container for more information.

Install Docker

To install Docker Community Edition for Windows:
Tech/SME Note to MemSQL: Can you use the legacy Docker Toolbox to install the MemSQL image?

  1. Navigate to the Docker Community Edition for Windows download page.
  2. Click “Please Login To Download.”
  3. On the “Welcome to Docker” page, login with your Docker ID or create a new account.
  4. If you created a new account, check your email and click the link provided to verify your email address. This should return you to the Docker Community Edition for Windows download page. Now that your account is verified, you’ll need to login to enable the download.
  5. Once you’re logged in, click “Get Docker” to download the installation file from the Docker Community Edition for Windows download page.
  6. After the installation file downloads, run “Docker for Windows Installer.exe” from your browser.

By default, Docker will create a separate environment on your PC that’s similar to a Linux virtual machine. This environment is automatically assigned a unique IP address, and is accessible from the Windows command prompt and your Windows 10 browser.
