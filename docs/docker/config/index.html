Docker Settings

To run the MemSQL image, Docker needs to have access to a minimum of 4 CPU cores and 4GB of RAM on your PC. To update your Docker settings:

  1. Right click the Docker icon from your system tray, then click Settings.
  2. Click Advanced. Set Docker’s CPUs to 4, and increase the memory to at least 4096MB (required). Then click Apply.

    Make sure to leave enough resources for Windows. Otherwise, you may experience out of memory exceptions and slow performance when you’re running Docker containers. Refer to Docker’s Limit a container’s resources for more information.

To configure additional settings, refer to Docker’s Docker Settings dialog.
